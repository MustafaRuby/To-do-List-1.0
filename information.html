Use info

Hi! This is your own to do list app, here's how you can use it:

Event insertion:
To insert you write the details about the event you want to plan (if you don't insert any details, you will not be able to insert any event), then you can add a date to your plan, if you don't, it'll automatically set the date to the day you inserted it on.

To add an event to the list, you can click the "Add event" button, or click "Enter" while in the input or date feild.

List use:
To check events that you already completed, either click on the checkbox or the event itself.

To eliminate the events you checked you can either click the "Remove checked events" buttons or simply press the "Backspace" or "delete" key.

To check all the elements simply press "Ctrl+A".

To select an event, click anywhere on the event other than the details of the event, the date, the time, and the checkbox.

To clear the list click the "Clear list" button.

To backup, you can go to the backup page with the "B" button on the top of the page, you can go back to the list page with the "L" button beside it.

The backup page has two functionalities:
- Generate Backup: generates a json string, that can be inputted into the app to recover the current events in the app;
- Backup; Uses whatever input in the backup input feild, and repositions the backed up events.

To modify an event, you can double click any part of the event (detils, date or time) and it'll allow you to modify, esc doesn't save changes.
